## 1. 实现

三元组缓存 `(t−K, t, t+K)`(317,454 组,`triples_full_k10`):以概率 0.4,训练时**用模型自生成的上一帧**(cond0 短 ODE-10 生成 → 门控 dopp-warp + 位姿变换)构造草稿,GPU-NN 在线配对;其余分支用真实草稿 + 预存 OT。损失同 v2full(flow + 静态 λ0.1 + L_temp λ0.1),dim512/depth8。

## 4. G3 判定(如实,分维度)

**✅ 重磅正结果 1——warp 机制本身在规模上完胜 ego-only**:copy_dopp 链 9.26 vs copy_ego 13.34(**−31%@2.5s**,步步领先自 1.0s 起)。mini 上方向相反(6.15>5.29)——规模 val 含高速场景,动态占比高,**Doppler-warp 的价值随动态性增长**,proposal 主线 B 的核心机制(式 4)在 2.5s 视距上定量成立。

**✅ 重磅正结果 2——SS 显著改善长时距稳定性**:ss_dopp 的 CD 曲线后段变平(11.5→11.9→12.9)而 big_dopp 发散(10.6→14.3→14.0);2.5s 处 ss 反超 copy_ego(12.95<13.34);物理新鲜度全程 SS 最优(52.0% vs big 43.6% vs copy_dopp 12.4%,GT 65.1%)。

**❌ 未竟——桥式精修的几何税仍未消除**:生成链几何仍劣于纯 copy_dopp 链(12.95 vs 9.26)。每步 ~1.5-2m 的精修代价在 5 步内未被 SS 完全治愈。

**G3 结论**:按判据字面("Doppler-warp + 时序损失优于 ego-only 下限")——**warp 机制维度已通过**(copy_dopp −31%);"生成精修"维度部分通过(物理新鲜度完胜 + SS 后几何追平 ego-only,但不及纯 warp 链)。**方法叙事据此定型:门控 Doppler-warp 是时序骨架(几何),桥式生成是物理刷新器(每步重建与当前 ego 自洽的 Doppler 场)——两者各司其职,合成完整的 FlowRadar 方案。**
