## 3. 自洽性校验(G0 硬判据)✅ 通过

方法:mini 全部 400 sample × 6 雷达(过滤 \|v_ego\|<2 m/s 与近距 <1 m),N=804,823 点;预测静态 `v_r = −(Rᵀv_ego)·r̂`,量测 `v_r = vrel·r̂`。

| 指标 | 结果 |
|------|------|
| H_raw 内点率(\|res\|<0.5 m/s) | **69.6%**(其余为动态点+噪声,符合预期) |
| H_comp 内点率 | 4.5% → 排除"已补偿"口径 |
| res_raw | **med −0.007 m/s,MAD 0.242 m/s** |
| 斜率(量测 vs 预测,\|pred\|>1) | 0.90–0.94(六通道一致) |
| 符号/坐标约定 | 与 proposal 公式一致,无需翻转 |

残差直方图:[assets/truckscenes_mini_doppler_consistency.png](assets/truckscenes_mini_doppler_consistency.png)(res_raw 在 0 处尖峰);完整输出:[assets/truckscenes_mini_field_report.txt](assets/truckscenes_mini_field_report.txt)。

**结论**:主线 A 的"静态解析硬约束"与主线 B 的 `v_r·Δt ≈ Δrange` 在 TruckScenes 上物理前提成立;斜率略小于 1 与 30% 外点来自动态目标/多径,正是生成模型需要物理约束的空间——可直接引用为动机数据。
